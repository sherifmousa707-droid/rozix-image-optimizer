# Rozix Image Optimizer

Rozix Image Optimizer is a free Windows desktop application developed by Rozix Systems.

## Features

- Convert images between multiple formats
- Compress image size
- Resize images in bulk
- Support for JPG, PNG, WebP, BMP, TIFF, HEIC, and ICO
- Arabic user interface
- Drag and Drop support
